## Schema (locked at packages/schemas/src/reports/v1.ts — DO NOT INVENT FIELDS)

GET /api/dashboard/reports?window=THIS%20WEEK|THIS%20MONTH|THIS%20QUARTER returns ReportDataV1:

{
label: "THIS WEEK" | "THIS MONTH" | "THIS QUARTER",
period: "APR 1 — APR 30", // human-readable date span
dateFolio: "APR 1 — APR 30", // typographic period anchor
pullquote: { // LLM-generated narrative
pre: string, value: string,
mid: string, cost: string,
post: string
},
attribution: { // who drove revenue
total: number, // SGD cents
delta: { kind: "pos"|"neg"|"flat", text: "↑ 18%" } | null,
riley: { value: number, caption: string }, // ad-driven leads converted
alex: { value: number, caption: string } // reply conversions
},
funnel: [ // 5 stages — exact order
{ stage: "Impressions", n: number, label: string, delta: Delta|null },
{ stage: "Clicks", n: number, label: string, delta: Delta|null },
{ stage: "Landing visits", n: number, label: string, delta: Delta|null },
{ stage: "Leads", n: number, label: string, delta: Delta|null },
{ stage: "Bookings", n: number, label: string, delta: Delta|null }
],
funnelNarrative: { marker: "Riley · Apr 22", text: string },
campaigns: CampaignRow[], // table; see fields below
cost: { paid: number, alt: number, saving: number }, // SGD cents
costNarrative: string, // e.g. "vs. SDR + agency"
managedComparison: ManagedComparisonData | null
}

CampaignRow: name, spend, impressions, inlineLinkClicks, costPerInlineLinkClick, inlineLinkClickCtr, leads, revenue, cpl (nullable), clickToLeadRate (nullable), roas.

ManagedComparisonData (when present): cohort comparison of managed vs unmanaged across ads & conversations — { ads: { spend, revenue, roas }, conversations: { replies, conversionRate, replyMinutesP50 } }.

## Currency

Backend currency is SGD (despite fmtMoney() in code hardcoding "$"). Format all values as S$X,XXX (no decimals over S$100; cents below). Build fmtSGD() helper. Do NOT use $ alone.

## Window control

Top-right: segmented control "THIS WEEK / THIS MONTH / THIS QUARTER" — default THIS MONTH. Selection drives ?window= query param. Add a small "refresh" button → POST /api/dashboard/reports/refresh?window=... (invalidates 1-hour cache and recomputes). Label: "Recompute (cached 47m ago)".

## Layout — editorial folio, top to bottom

1. Header
   - Tiny section label "STATEMENT" upper-left in 11px tracked
   - Display title "Operator's Statement" in Instrument Sans
   - Right: dateFolio in mono ("APR 1 — APR 30") + window selector beneath

2. Pull Quote
   - Large display text composed from pullquote fields: `{pre}<value>{mid}<cost>{post}` where <value> and <cost> are emphasized (font-weight + amber accent on hairline underline)
   - Max ~24rem wide; centered or left-aligned with side rules

3. Attribution
   - Single hero number: total in SGD, display font, very large
   - Delta badge inline (pos = subtle amber up-arrow, neg = subtle down, flat = em-dash); NO red/green
   - Two split cards beneath in a 50/50 grid:
     - Riley card: value + caption (e.g., "ad-driven leads converted")
     - Alex card: value + caption (e.g., "reply conversions")
   - Cards are hairline-bordered, low-contrast, mono numerics

4. Funnel
   - Custom SVG (no chart library) — 5 horizontal bars stacked, each row: stage label (left), proportional bar (middle, fills based on n/max(funnel.n)), label string (right, mono), delta chip if present
   - Beneath the bars: funnelNarrative — "Riley · Apr 22" marker styled like an editorial byline, then text in Inter italic
   - First-bar-100% pattern is FINE (Impressions is the top of funnel)

5. Campaigns table
   - Columns: Campaign, Spend, Impressions, Clicks (with CTR caption beneath), CPC, Leads, CPL, Revenue, ROAS, click→lead rate
   - Sticky first column on horizontal scroll
   - Sortable headers (default sort: revenue desc)
   - Mono numerics, hairlines for row separators
   - Footer totals row in mono
   - Mobile fallback: card list, not horizontal scroll
   - Highlight ROAS by depth of amber underline (calibrated; never RGB)

6. Cost vs Value
   - Three-cell horizontal layout:
     - LEFT: "You pay" — `cost.paid` (SGD)
     - MIDDLE: "SDR + agency alternative" — `cost.alt`
     - RIGHT: "Monthly saving" — `cost.saving` (largest, amber emphasis)
   - Beneath: `costNarrative` in italic
   - This is the renewal-checkpoint moment; design it like the punchline of the page

7. Managed Comparison (only when managedComparison !== null)
   - Section label "MANAGED VS UNMANAGED"
   - Two columns: Ads block (spend / revenue / ROAS) and Conversations block (replies / conversion rate / median reply minutes)
   - Each metric: number in mono, label below, delta if available
   - Hide gracefully when null — no empty placeholder card

8. Footer
   - Section label "COLOPHON"
   - Period restated, generation timestamp, "Live data" / "Sample data" badge driven by NEXT_PUBLIC_REPORTS_LIVE
   - Caveat micro-copy in --sw-text-muted

## Live vs fixture mode

NEXT_PUBLIC_REPORTS_LIVE=false → render from FIXTURES_BY_WINDOW:

- THIS WEEK → quietFixture (low numbers)
- THIS MONTH → goodFixture (the showcase, S$14,700ish)
- THIS QUARTER → problemFixture (mixed signals)
  NEXT_PUBLIC_REPORTS_LIVE=true → fetch GET /api/dashboard/reports?window=…

Connections required for non-zero live data: Meta Ads `Connection` (status=connected, accessToken, externalAccountId) and Stripe plan in org config and populated revenue_events/bookings.

When connections are missing in live mode, show a calm banner above the attribution block: "No Meta Ads connection — campaigns and funnel will read zero. Connect under Settings."

## Design system

- --sw-base #F5F3F0 background
- --sw-surface raised cards
- --sw-accent #A07850 — amber underlines, delta-up arrows, ROAS depth
- Display: Instrument Sans (title, hero number, pull-quote value/cost)
- Body: Inter
- Mono: JetBrains Mono for ALL numbers
- Hairline borders, no shadows
- Generous vertical rhythm — this is a statement, not a SaaS dashboard
- page-width 74rem

## Anti-patterns

- No green-good / red-bad. Delta direction is conveyed by glyph + subtle amber depth.
- No chart library — funnel and attribution are custom SVG / grid.
- No USD `$`. SGD only. Format as S$X,XXX.
- No "trending widgets," no sparklines glued onto every number.
- No emoji status. Statement-grade typography only.
- Don't omit managedComparison when present; don't show a card when null.
