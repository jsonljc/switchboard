# Switchboard — Hermes Project Context

## This repo

TypeScript monorepo. pnpm workspaces. Turborepo builds.
Run tests: `pnpm test` from root, or `pnpm test --filter @switchboard/agents` for a package.
Run typecheck: `pnpm typecheck`
Lint: `pnpm lint`

## Commit conventions

Conventional commits: feat/fix/refactor/docs/test/chore/perf(scope): description
Scope = package name without @switchboard/ prefix (e.g. `feat(agents): ...`)

## Branch conventions

feat/description, fix/description, refactor/description

## Never

- Push to main or master directly
- Merge PRs without CI passing
- Skip the sealed approval pattern for any action that mutates state
- Modify packages/schemas without updating all consumers

## Cartridge pattern

Every cartridge implements the CartridgeInterface from packages/cartridge-sdk.
Always check packages/cartridge-sdk/src/types.ts before writing a new cartridge.
Tests live in packages/cartridge-sdk/src/**tests**/ for SDK-level tests,
and in cartridges/<name>/src/**tests**/ for cartridge-specific tests.
